# 快速开始指南

## 5分钟快速上手

### 1. 安装依赖

Windows:
```bash
install.bat
```

Linux/Mac:
```bash
chmod +x install.sh
./install.sh
```

或手动安装:
```bash
pip install -r requirements.txt
```

### 2. 启动系统

Windows:
```bash
run.bat
```

或直接运行:
```bash
python main.py
```

### 3. 登录系统

使用以下测试账号登录：

| 角色 | 用户名 | 密码 |
|------|--------|------|
| 管理员 | admin | admin123 |
| 教师 | teacher1 | 123456 |
| 学生 | student1 | 123456 |

## 教师使用流程

### 创建作业

1. 登录教师账号
2. 点击"作业管理"标签
3. 点击"新建作业"按钮
4. 填写作业信息：
   - 标题：例如"第一章测试"
   - 描述：作业说明
   - 截止时间：格式 YYYY-MM-DD HH:MM

### 添加题目

1. 在作业列表中双击作业
2. 点击"添加单选题"或"添加主观题"
3. 填写题目信息：
   - 题目内容
   - 标准答案
   - 题目解析（主观题AI评分参考）
   - 分值

### 查看提交

1. 在作业详情页面查看学生提交
2. 可以手动调整主观题分数

### 学情分析

1. 点击"学情分析"标签
2. 选择作业查看统计图表
3. 查看平均分、及格率等数据

## 学生使用流程

### 完成作业

1. 登录学生账号
2. 点击"我的作业"标签
3. 双击未完成的作业
4. 依次回答所有题目
5. 点击"提交作业"

### 查看成绩

1. 点击"历史提交"标签
2. 查看已提交作业的成绩和反馈

## 常见问题

### Q: 忘记密码怎么办？
A: 目前系统不支持密码重置，请联系管理员或直接修改数据库。

### Q: 如何修改默认账号？
A: 编辑 `start.sql` 文件，删除数据库后重新运行程序。

### Q: 主观题评分不准确？
A: AI评分仅供参考，教师可以手动调整分数。建议在题目解析中提供详细的评分标准。

### Q: 如何备份数据？
A: 复制 `data/teaching_assistant.db` 文件即可。

### Q: 系统支持多少学生？
A: SQLite 理论上支持数千用户，但建议小班教学使用（<100人）。
