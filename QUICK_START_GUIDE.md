# 快速启动指南

## 系统要求
- Python 3.8+
- Windows 10/11 (推荐)
- 2GB+ 可用内存
- 500MB+ 可用磁盘空间

## 快速启动步骤

### 方法一：使用批处理脚本（推荐）

#### Windows用户
1. 双击运行 `install.bat` 安装依赖
2. 双击运行 `run.bat` 启动系统

### 方法二：手动启动

#### 1. 安装依赖
```bash
# 激活虚拟环境（如果有）
.venv\Scripts\activate

# 安装所需包
pip install -r requirements.txt
```

#### 2. 启动系统
```bash
python main.py
```

## 首次使用指南

### 教师端操作流程
1. **登录系统**
   - 使用教师账户登录

2. **创建班级**
   - 点击"班级管理" → "创建班级"
   - 填写班级名称、描述、人数限制
   - 生成班级代码（学生用于加入）

3. **创建课程**
   - 点击"课程管理" → "创建课程"
   - 填写课程标题、描述
   - 关联班级（可选）

4. **发布课程**
   - 在课程列表中选择课程
   - 点击"发布"按钮

5. **创建作业**
   - 点击"作业管理" → "新建作业"
   - 填写作业标题、描述、截止时间
   - 选择关联课程和班级

6. **批改作业**
   - 点击"成绩管理"
   - 选择课程和作业
   - 查看学生提交，录入成绩

7. **查看数据分析**
   - 点击"数据分析"
   - 查看成绩分布、完成率等统计

### 学生端操作流程
1. **登录系统**
   - 使用学生账户登录

2. **浏览课程**
   - 点击"我的课程"
   - 查看可选课程列表

3. **选课**
   - 选择感兴趣的课程
   - 点击"选课"按钮

4. **查看作业**
   - 点击"我的作业"
   - 查看待完成、已提交的作业

5. **提交作业**
   - 选择作业
   - 点击"提交作业"
   - 填写答案并提交

6. **查看成绩**
   - 点击"我的成绩"
   - 选择课程查看详细成绩

7. **参与讨论**
   - 点击"讨论区"
   - 发帖提问或回复他人

## 常见问题

### Q1: 系统无法启动？
**A:** 检查以下几点：
1. 确认已安装 Python 3.8+
2. 确认已安装所有依赖包：`pip install -r requirements.txt`
3. 检查数据库文件是否存在：`data/teaching_assistant.db`
4. 查看日志文件：`logs/app.log`

### Q2: 登录失败？
**A:** 
1. 确认使用正确的用户名和密码
2. 检查数据库中是否有用户数据
3. 尝试重新初始化数据库：运行 `start.sql`

### Q3: 界面显示异常？
**A:**
1. 确认已安装 ttkbootstrap：`pip install ttkbootstrap`
2. 尝试更新 tkinter
3. 检查屏幕分辨率设置

### Q4: 中文显示乱码？
**A:**
1. 系统已配置中文字体支持
2. 确认系统安装了 SimHei 或 Microsoft YaHei 字体
3. 检查 matplotlib 配置

### Q5: 如何添加新用户？
**A:**
1. 使用管理员账户登录
2. 进入用户管理界面
3. 点击"添加用户"按钮
4. 填写用户信息并保存

### Q6: 如何备份数据？
**A:**
1. 复制 `data/teaching_assistant.db` 文件
2. 保存到安全位置
3. 恢复时替换该文件即可

### Q7: 如何重置密码？
**A:**
1. 使用管理员账户登录
2. 进入用户管理
3. 选择用户，点击"重置密码"

### 数据库优化

- 关闭不必要的后台进程

## 安全建议

1. **定期更改密码**
   - 建议每3个月更改一次密码
   - 使用强密码（8位以上，包含字母、数字、符号）

2. **数据备份**
   - 每周备份一次数据库
   - 保存多个备份版本

3. **访问控制**
   - 不要共享管理员账户
   - 及时删除离职人员账户

4. **系统更新**
   - 及时更新系统到最新版本
   - 关注安全公告
