# 系统演示指南

## 快速演示流程（5分钟）

### 准备工作

1. 安装依赖
```bash
pip install -r requirements.txt
```

2. 生成测试数据
```bash
python scripts/generate_test_data.py
```

3. 启动系统
```bash
python main.py
```

## 演示场景

### 场景一：教师创建作业（2分钟）

1. **登录教师账号**
   - 用户名：`teacher1`
   - 密码：`123456`

2. **创建新作业**
   - 点击"作业管理"标签
   - 点击"新建作业"按钮
   - 填写信息：
     - 标题：Python 基础测试
     - 描述：测试 Python 基础知识
     - 截止时间：2024-12-31 23:59

3. **添加题目**
   - 双击刚创建的作业
   - 点击"添加单选题"
   
   **单选题示例：**
   ```
   题目内容：Python 是什么类型的语言？
   A. 编译型
   B. 解释型
   C. 汇编语言
   D. 机器语言
   
   标准答案：B
   分值：10
   解析：Python 是一种解释型的高级编程语言
   ```
   
   - 点击"添加主观题"
   
   **主观题示例：**
   ```
   题目内容：请简述 Python 列表和元组的区别
   
   标准答案：列表是可变的，可以修改元素；元组是不可变的，创建后不能修改。列表使用方括号[]，元组使用圆括号()。
   
   分值：20
   解析：主要考察可变性、语法、使用场景等方面
   ```

### 场景二：学生完成作业（2分钟）

1. **注销并登录学生账号**
   - 点击右上角"注销"
   - 用户名：`student1`
   - 密码：`123456`

2. **查看作业列表**
   - 点击"我的作业"标签
   - 查看可用作业

3. **完成作业**
   - 双击"Python 基础测试"
   - 回答问题：
     - 单选题：输入 `B`
     - 主观题：输入 `列表可以修改，元组不能修改`
   - 点击"提交作业"

4. **查看成绩**
   - 点击"历史提交"标签
   - 查看刚才提交的成绩和 AI 反馈

### 场景三：教师查看统计（1分钟）

     - 平均分
     - 及格率

3. **手动调整分数（可选）**
   - 返回"作业管理"

## 常见演示问题

### Q: AI 评分准确吗？
A: AI 评分基于文本相似度，对于标准答案明确的题目准确率较高。对于开放性问题，建议教师手动审核调整。系统提供了详细的评分反馈，教师可以根据实际情况调整分数。

### Q: 支持多少学生？
A: 系统基于 SQLite，理论上支持数千用户。建议小班教学使用（<100人）。如需支持更多用户，可以切换到 MySQL 或 PostgreSQL。

### Q: 如何备份数据？
A: 只需复制 `data/teaching_assistant.db` 文件即可。建议定期备份，确保数据安全。

### Q: 可以导出成绩吗？
A: 系统支持导出为 Excel 和 PDF 格式。可以使用 `utils/export_utils.py` 中的工具函数。

### Q: 如何添加新功能？
A: 系统采用模块化设计，易于扩展。可以在 `modules/` 目录下添加新的服务模块，在 `ui/` 目录下添加界面组件。详细开发指南请查看 `docs/DEVELOPMENT.md`。
